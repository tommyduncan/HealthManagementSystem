壹、什麼是糖尿病？


  • 正常情況下，身體會將吃進去的澱粉類食物轉變成葡萄糖，作為身體的燃料。而胰臟會分泌一種荷爾蒙叫做『胰島素』，它可以幫助葡萄糖進入細胞內，提供熱能。糖尿病人因為胰臟不能製造足夠的胰島素，葡萄糖無法充分進入細胞內，血糖濃度就會升高，形成糖尿病。

貳、誰易得糖尿病？


  一、近親有糖尿病的人會比一般人高出5倍以上機率

  二、40歲以上的中、老年人

  三、體型肥胖的人

叁、糖尿病的症狀


  一、三多症狀 : 吃多 、喝多、尿多

  二、體重減少

  三、疲倦

  四、傷口不易癒合

  五、皮膚搔癢

肆、糖尿病的診斷(經抽血測血糖並由醫師診斷)


   一、一次以上空腹血糖值大於126mg/dl

  二、不論空腹與否，血糖值大於200mg/dl

伍、糖尿病的慢性合併症


  • 腦血管病變、神經病變
  • 眼睛病變
  • 心臟病
  • 腎臟病變
  • 足部循環不良

陸、糖尿病的治療 (應遵從醫療人員的指示)


   一、飲食控制

   二、規律及適度運動

   三、與醫師配合正確用藥

   四、控制自己的體重在理想範圍 :
    理想體重的計算方式有兩種
    (一).男 = (身高-80 ) x 0.7
           女 = (身高-70 ) x 0.6
    (二).身高(公尺)² x 22 加減10%內皆算理想範圍


柒、糖尿病控制良好的指標


   一、空腹血糖：80-120mg/dl

   二、飯後兩小時血糖：80-160mg/dl

   三、糖化血色素：4-6 ﹪



糖尿病人只要能遵守糖尿病的保健之道，做到良好的控制，學習如何與糖尿病共同生活，是可以和一般人一樣享有正常的生活。
